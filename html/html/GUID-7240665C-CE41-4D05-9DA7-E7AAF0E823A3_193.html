即便成功安装后，也无法在网络中找到我的 Fuji Xerox 设备。

相关型号:DocuPrint P268 d / DocuPrint P268 dw
原因 措施 接口
您的安全软件阻止设备访问网络。

一些安全软件可能会阻止访问不显示安全警报对话框，即使成功安装后也是如此。

若要允许访问，请参见安全软件附带的说明书或咨询制造商。

有线/无线
分配给本 Fuji Xerox 设备的 IP 地址不可用。

  • 确认 IP 地址和子网掩码。

    验证计算机和 Fuji Xerox 设备的 IP 地址和子网掩码是否正确，且位于相同网络内。

    关于如何验证 IP 地址和子网掩码，请咨询网络管理员。

有线/无线
您的 Fuji Xerox 设备尚未使用无线功能连接到网络。

打印 WLAN 报告，检查打印报告上的错误代码。

无线

如果您已确认并尝试以上所有方法，但仍无法在网络中找到您的 Fuji Xerox 设备，请卸载打印驱动程序，然后重新安装。
