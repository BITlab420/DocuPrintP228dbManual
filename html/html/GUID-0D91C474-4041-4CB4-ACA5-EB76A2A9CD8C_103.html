什么是网络基本管理？

相关型号:DocuPrint P268 d / DocuPrint P268 dw

网络基本管理实用程序使用标准网络浏览器通过超文本传输协议 (HTTP) 对您的设备进行管理。将设备的 IP 地址输入至网络浏览器以访问和更改打印服务器设置。

  • 我们建议 Windows® 用户使用 Microsoft® Internet Explorer® 8.0/9.0/10.0，Macintosh 用户使用 Safari 5.0/6.0。无论使用何种浏览器，请确保始终启用 JavaScript 和 Cookies。如果使用其他网络浏览器，请确保其兼容 HTTP 1.0 和 HTTP 1.1。

  • 您必须在网络中使用 TCP/IP 协议，并为打印服务器和计算机指定有效的 IP 地址。
