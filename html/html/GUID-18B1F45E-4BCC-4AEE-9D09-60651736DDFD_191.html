安装打印机驱动程序期间，无法在网络中找到我的 Fuji Xerox 设备。

相关型号:DocuPrint P268 d / DocuPrint P268 dw
原因
措施
接口
您的计算机未正确连接到网络。
确保计算机已连接至网络，即局域网环境或互联网服务。有关网络问题方面的支持，请联系您的网络管理员。
有线/无线
重新将网络接口电缆连接至集线器，确保网络接口电缆和网络连接状况良好。如果可能，尝试用其他电缆将设备连接到集线器的其他端口。
有线
您的设备未正确连接到网络，或您的设备没有有效的 IP 地址。
打印网络配置报告，并确认 Ethernet Link Status（以太网连接状态) 或 Wireless Link Status 显示为 Link OK(连接良好)。

如果没有显示 Link OK(连接良好)，请执行以下操作：

  • 确保有线或无线设置已激活。在报告顶端检查此设置。

  • 咨询网络管理员您的 IP 地址是否有效。
  • 确认您的无线局域网接入点/路由器可用。

有线/无线
您的安全软件阻止设备访问网络。
  • 在安装程序对话框中，选择选项，再次搜索 Fuji Xerox 设备。
  • 打印机安装程序安装过程中出现安全软件的警报信息时，允许访问。
  • (Macintosh)

    如果您正在使用反间谍软件或防病毒软件的防火墙功能，请暂时将其禁用，然后安装 Fuji Xerox 软件。

有线/无线
您的 Fuji Xerox 设备放置在离无线局域网接入点/路由器过远的地方。
配置无线网络设置时，请将 Fuji Xerox 设备放置在离无线局域网接入点/路由器大约 1 米范围内。
无线
您的设备和无线局域网接入点/路由器之间有障碍物 (例如墙壁或家具)。
将 Fuji Xerox 设备移至没有障碍或者更加靠近无线局域网接入点/路由器的地方。
无线
Fuji Xerox 设备或无线局域网接入点/路由器附近有无线计算机、蓝牙设备、微波炉或数字无绳电话。
将所有这些设备搬离 Fuji Xerox 设备或无线局域网接入点/路由器。
无线
