移动打印 Fuji Xerox 打印和扫描实用程序

相关型号:DocuPrint P268 d / DocuPrint P268 dw

此打印机支持通过各种使用 Fuji Xerox 打印机和扫描实用程序 (Small Office (iOS / Android)) 的移动设备进行打印

如果设备没有扫描器，则无法使用 Fuji Xerox 打印机和扫描实用程序的扫描功能。

对于 Android™ 设备

从 Google Play™ 免费下载或者请访问网站在线支持：

http://www.fxap.com.sg/product/software/printscanutilandroid/index.jsp

适用于 iOS 设备

从 App Store 免费下载或者请访问网站在线支持：

http://www.fxap.com.sg/product/software/printscanutilios/index.jsp

对于 Windows Phone® 设备

从 Windows Phone® Store 下载并安装 Fuji Xerox 打印机和扫描实用程序。为能充分利用 Fuji Xerox 打印机和扫描实用程序支持的所有功能，请确保您使用的是最新版本。
