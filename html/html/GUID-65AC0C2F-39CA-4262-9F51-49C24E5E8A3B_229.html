设置继续运行模式

开启继续运行模式时，即使操作面板显示需要更换墨粉盒，也将强制设备继续打印。显示更换墨粉盒错误信息时，除非更换墨粉盒，否则无法开启此模式。

我们建议您尽快更换墨粉盒。继续运行模式开启时打印效果可能会模糊不清；不保证打印质量。出厂设置为关。

DocuPrint P268 b/DocuPrint P228 db
DocuPrint P268 d/DocuPrint P268 dw

DocuPrint P268 b/DocuPrint P228 db

  1. 确保已合上前盖且已插入电源线插头。
  2. 按 打开设备电源并确保仅 就绪 指示灯亮起。
  3. 按执行 七次开启或关闭此功能。

    开启继续运行模式时，所有指示灯闪烁两次。

    关闭继续运行模式时，所有指示灯闪烁一次。

DocuPrint P268 d/DocuPrint P268 dw

  1. 按 或 选择General Setup (常规设置)选项，然后按 确定。
  2. 按 或 选择Replace Toner (更换墨粉盒)选项，然后按确定。
  3. 按 或 选择Continue (继续)或Stop (停止)选项。
    按执行。
