出厂设置

使用出厂设置功能可将某些设备设置重置为原始出厂设置。
DocuPrint P268 b/DocuPrint P228 db
DocuPrint P268 d/DocuPrint P268 dw

DocuPrint P268 b/DocuPrint P228 db

  1. 按住 关闭设备电源。
  2. 确保已合上前盖且已插入电源线插头。
  3. 按 打开设备电源的同时，按住执行。按住执行，直到 墨粉、感光鼓 和 纸张 指示灯亮起。
  4. 按执行八次。
    确保所有四个指示灯全部亮起，表示设备已经重置为出厂默认设置 (接口和语言设置除外)。

DocuPrint P268 d/DocuPrint P268 dw

  1. 按 或 选择Reset Menu (重置菜单)选项，然后按确定。
  2. 按 或 选择Factory Reset (出厂重置)选项，然后按确定。
  3. 按 选择Reset (重置)。
