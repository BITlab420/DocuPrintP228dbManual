IP 地址

互联网协议 (IP) 地址是识别连接到网络的每个设备的一系列号码。IP 地址包含以小数点分隔的四个号码。每个号码均位于 0 至 255 之间。

例如：在小型网络中，您通常会更改最后一个号码。

192.168.1.1
192.168.1.2
192.168.1.3

如果将 IP 地址指定给您的打印服务器：

如果网络中有 DHCP/BOOTP/RARP 服务器，打印服务器将从该服务器自动获取其 IP 地址。

在小型网络中，DHCP 服务器还可用作路由器。

如果网络中没有 DHCP/BOOTP/RARP 服务器，自动专用 IP 寻址 (APIPA) 协议将在 169.254.1.0 至 169.254.254.255 范围内自动指定 IP 地址。
