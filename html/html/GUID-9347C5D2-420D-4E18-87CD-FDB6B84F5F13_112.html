配置 Fuji Xerox 设备无线网络的其他方法

若要将设备连接到无线网络，我们建议使用随机安装光盘。
  • 为接入无线网络配置您的 Fuji Xerox 设备之前
  • 使用 USB 接口电缆配置设备的无线网络
    使用 USB 接口电缆将本 Fuji Xerox 设备连接到您的无线网络。一旦连接完成，移除 USB 接口电缆。
  • 无需 USB 接口电缆，使用 Fuji Xerox 安装应用程序配置设备的无线网络
    通过无线方式将本 Fuji Xerox 设备添加到您的无线网络中。
  • 使用 Wi-Fi Protected Setup™ (WPS) 或 AOSS™ 配置设备的无线网络
  • 使用 Wi-Fi Protected Setup™ (WPS) 的 PIN 方式配置设备的无线网络
    如果您的无线局域网接入点/路由器支持 WPS，可使用个人识别码 (PIN) 方式配置您的无线网络设置。
  • 在 Ad-Hoc 模式下配置设备的无线网络 (适用于 IEEE 802.11b/g/n)
  • 使用设备操作面板安装向导配置设备的无线网络
  • 不多址发送 SSID 时配置设备的无线网络
  • 使用 Wi-Fi Direct™
