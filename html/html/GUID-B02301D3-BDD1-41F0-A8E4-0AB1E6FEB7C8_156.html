TCP/IP

相关型号:DocuPrint P268 d / DocuPrint P268 dw
如果您使用网络电缆将设备连接到网络，请使用Wired LAN (有线局域网)菜单选项。如果您将设备连接到无线以太网，请使用WLAN (无线网络)菜单选项。

引导方式

此选项控制设备如何获取 IP 地址。

自动

在此模式中，设备将扫描网络寻找 DHCP 服务器。如果可以找到，且 DHCP 服务器配置为向设备分配 IP 地址，那么将使用 DHCP 服务器所提供的 IP 地址。

如果没有 DHCP 服务器，将使用 APIPA 协议设置 IP 地址。设备首次打开电源后，将耗时几分钟扫描网络寻找服务器。

静态

在此模式中，设备的 IP 地址必须手动指定。输入 IP 地址时，将锁定为指定地址。

若不想通过 DHCP、BOOTP 或 RARP 配置打印服务器，必须将引导方式设置为静态，使打印服务器有一个静态的 IP 地址。这样可以防止打印服务器试图从这些系统中获取 IP 地址。

若要更改引导方式，请使用设备操作面板、BRAdmin Light 或网络基本管理。

IP 地址

此字段显示设备当前的 IP 地址。如果选择了静态引导方式，请输入您想分配给设备的 IP 地址 (向网络管理员查询要使用的 IP 地址)。如果选择了静态以外的方式，设备将会尝试使用 DHCP、BOOTP 或 RARP 协议确定其 IP 地址。

设备的默认 IP 地址可能会与网络的 IP 地址编号方案不符。建议联系您的网络管理员，获取与计算机连接网络相符的 IP 地址。

子网掩码

此字段显示设备当前使用的子网掩码。若不使用 DHCP、BOOTP 或 RARP 获取子网掩码，请输入所需的子网掩码。请与网络管理员联系，确认要使用的子网掩码。

网关

此字段显示设备当前使用的网关或路由器地址。若不使用 DHCP、BOOTP 或 RARP 获取网关或路由器地址，请输入您想要分配的地址。若无网关或路由器，将此字段留空。若不确定，请与网络管理员联系确认。

尝试 IP 启动

此字段显示当引导方式设为静态以外的其他方式时设备尝试获取 IP 地址的次数。

APIPA

打印服务器无法从您设置的引导方式获取 IP 地址时，若选择On (开)，打印服务器将在 169.254.1.0 - 169.254.254.255 范围内自动分配链接本地 IP 地址。打印服务器无法从您设置的引导方式获取 IP 地址时，若选择Off (关)，则表明 IP 地址保持不变。

IPv6

本设备与下一代互联网协议 IPv6 兼容。默认设置为Off (关)。如果您想使用 IPv6 协议，请选择On (开)。

  • 如果您将IPv6设置为On (开)，请关闭电源开关，然后再将其打开以启用此协议。
  • 将IPv6设为On (开)后，此设置将应用到有线和无线 LAN 接口。
