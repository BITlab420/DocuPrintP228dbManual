设置重置

DocuPrint P268 b/DocuPrint P228 db
DocuPrint P268 d/DocuPrint P268 dw

DocuPrint P268 b/DocuPrint P228 db

使用设置重置功能可将所有设备设置重置为原来的出厂设置。

当要处理掉本设备时，Fuji Xerox 强烈建议您执行此操作。

执行设置重置前，拔下所有接口电缆。
  1. 按住 关闭设备电源。
  2. 确保已合上前盖且已插入电源线插头。
  3. 按 打开设备电源的同时，按住执行。按住执行，直到 墨粉、感光鼓 和 纸张 指示灯亮起。
  4. 松开执行。确保所有指示灯熄灭。
  5. 按执行十次。
    设备将自动重新启动。

DocuPrint P268 d/DocuPrint P268 dw

  1. 断开网络电缆与设备的连接，否则将无法重置 IP 地址等网络设置。
  2. 按 或 选择Reset Menu (重置菜单)选项，然后按确定。
  3. 同时按 和返回。
  4. 显示Settings Reset (设置重置)时，按 确定。
  5. 显示Restart Printer? (重启打印机？)时，按 确定。
    设备将自动重新启动。
