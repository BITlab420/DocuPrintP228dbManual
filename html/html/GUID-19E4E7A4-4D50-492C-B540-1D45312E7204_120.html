如何使用 Wi-Fi Protected Setup™ (WPS) 的 PIN 方式配置无线网络

相关型号:DocuPrint P268 dw
  1. 按 或 选择Network (网络)选项，然后按 确定。
  2. 按 或 选择WLAN (无线网络)选项，然后按 确定。
  3. 按 或 选择WPS w/PIN Code (有 PIN 码的WPS)选项，然后按确定。
  4. 显示WLAN Enable ? (启用无线局域网？)时，按 选择开。
    此操作将启动无线安装向导。要取消，请按 返回。
  5. LCD 显示一个 8 位数的 PIN 码，然后设备开始搜索无线局域网接入点/路由器。
  6. 使用联网的计算机，在您的浏览器中输入 "http://access point's IP address" (其中，"access point's IP address" 为用作注册器的设备的 IP 地址)。
  7. 请转到 WPS 设置页面并输入 PIN 码，然后按照屏幕提示执行操作。
    • 注册器通常指无线局域网接入点/路由器。
    • 根据无线局域网接入点/路由器的品牌不同，设置页面也会有所不同。更多信息，请参见您的无线局域网接入点/路由器附带的说明书。
如果您使用 Windows Vista®、Windows® 7 或 Windows® 8 计算机作为注册器，请完成以下步骤：
  1. 执行以下操作中的一项：
    • (Windows Vista®)

      点击 (开始)>网络>添加无线设备。

    • (Windows® 7)

      点击 (开始)>设备和打印机>添加设备。

    • (Windows® 8)

      将鼠标移到桌面的右下角。显示菜单栏时，点击设置>控制面板>硬件和声音>设备和打印机>添加设备。

    • 若要使用 Windows Vista®、Windows® 7 或 Windows® 8 计算机作为注册器，必须事先将其注册到您的网络。更多信息，请参见您的无线局域网接入点/路由器附带的说明书。
    • 如果您使用 Windows® 7 或 Windows® 8 作为注册器，则可以在无线配置之后遵循屏幕提示安装打印机驱动程序。若要安装完整的驱动程序和软件包，使用随机安装光盘上的 Fuji Xerox 安装程序。
  2. 选择设备，然后点击下一步。
  3. 输入设备 LCD 上显示的 PIN 码，然后点击下一步。
  4. 选择网络，然后点击下一步。
  5. 点击关闭。
如果已成功连接无线设备，设备 LCD 上会显示Connected (已连接)。设备将打印一份无线连接状态报告。如果连接失败，请查看所打印报告上的错误代码。
如果连接失败，设备将还原到有线网络接口。

对于 Windows®：

无线网络设置完成。若要安装操作设备所必需的驱动程序和软件，在随机安装光盘的顶部菜单页面上点击安装打印机驱动程序。

对于 Macintosh：

无线网络设置完成。请转到 Fuji Xerox 网站 (http://www.fujixeroxprinters.com) 上您的设备型号的下载页面，以安装操作设备所需的驱动程序和软件。
