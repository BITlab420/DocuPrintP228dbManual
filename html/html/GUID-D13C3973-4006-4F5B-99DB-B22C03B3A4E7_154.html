无线局域网报告错误代码

相关型号:DocuPrint P268 dw
如果无线局域网报告显示连接失败，请检查打印报告上的错误代码，并参照下表中的相应指示：

错误代码

问题和建议解决方案

TS-01

无线设置未激活，请开启无线设置。

如果网络接口电缆连接到设备，请断开连接，再开启设备的无线设置。

TS-02

无法检测到无线接入点/路由器

  1. 检查以下四点：
    • 确保无线接入点/路由器电源已打开。
    • 将本设备移至没有障碍或者更加靠近无线接入点/路由器的地方。
    • 配置无线设置时，暂时将本设备放置在离无线局域网接入点大约 3.3 英尺 (1 米) 范围内。
    • 如果无线接入点/路由器使用 MAC 地址过滤，确认过滤器中允许使用 Fuji Xerox 设备的 MAC 地址。
  2. 如果手动输入 SSID 和安全信息 (SSID/验证方法/加密方法/网络密钥)，信息可能会不正确。

    确认 SSID 和安全信息，必要时输入正确信息。

TS-04

您的设备不支持所选无线接入点/路由器使用的验证方法/加密方法。

对于基础架构模式，更改无线接入点/路由器的验证方向和加密方法。您的设备支持以下验证方法：

  • WPA-Personal

    TKIP 或 AES

  • WPA2-Personal

    AES

  • 开放

    WEP 或 None (不加密)

  • 共享密钥

    WEP

如果仍然无法解决问题，您输入的 SSID 或网络设置可能不正确。配置无线网络设置。

对于 Ad-Hoc 模式，更改计算机无线设置的验证方向和加密方法。您的设备支持开放验证方法，可选 WEP 加密。

TS-05

您的安全信息 (SSID/网络密钥) 不正确。

确认 SSID 和网络密钥。如果您的路由器使用 WEP 加密，输入用作 WEP 密钥的密钥。Fuji Xerox 设备仅支持使用第一个 WEP 密钥。

TS-06

无线安全信息 (验证方法/加密方法/网络密钥) 不正确。

确认 TS-04 中所列的无线安全信息 (验证方法/加密方法/网络密钥)。如果您的路由器使用 WEP 加密，输入用作 WEP 密钥的密钥。Fuji Xerox 设备仅支持使用第一个 WEP 密钥。

TS-07

设备无法检查到启用 WPS 或 AOSS™ 的无线接入点/路由器。

如果使用 WPS 或 AOSS™ 配置无线设置，必须在设备和无线接入点/路由器上操作。确认无线接入点/路由器支持 WPS 或 AOSS™，然后再尝试启动。

如果您不知道如何使用 WPS 或 AOSS™ 操作无线接入点/路由器，请参见无线接入点/路由器随附文件，或咨询无线接入点/路由器制造商或您的网络管理员。

TS-08

检测到两个或多个启用 WPS 或 AOSS™ 的无线接入点。

  • 确认仅有一个范围内的无线接入点/路由器启用了 WPS 或 AOSS™，然后再试一次。
  • 几分钟后尝试重新启动，以避免来自其他接入点的干扰。
