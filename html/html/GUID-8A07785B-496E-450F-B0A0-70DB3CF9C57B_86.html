恢复打印机出厂设置

使用此功能可将设备的打印机设置恢复为出厂设置。

此操作无法重置网络设置。

DocuPrint P268 b/DocuPrint P228 db
DocuPrint P268 d/DocuPrint P268 dw

DocuPrint P268 b/DocuPrint P228 db

  1. 按住 关闭设备电源。
  2. 确保已合上前盖且已插入电源线插头。
  3. 按 打开设备电源的同时，按住执行。按住执行，直到 墨粉、感光鼓 和 纸张 指示灯亮起。
  4. 松开执行。确保所有指示灯熄灭。
  5. 按执行八次。确保所有指示灯全部亮起，表示设备已经重置为出厂默认设置 (接口和语言设置除外)。

DocuPrint P268 d/DocuPrint P268 dw

  1. 按 或 显示Reset Menu (重置菜单)选项，然后按确定。
  2. 按 或 显示Factory Reset (出厂重置)选项，然后按确定。
  3. 按 选择Reset (重置)。
