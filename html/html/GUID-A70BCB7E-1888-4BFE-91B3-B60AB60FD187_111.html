使用 BRAdmin Light 时访问网络设置 (Windows® 和 Macintosh)

相关型号:DocuPrint P268 d / DocuPrint P268 dw
  • 请转到 Fuji Xerox 网站 (http://www.fujixeroxprinters.com) 上您的设备型号的下载页面，下载最新版本的 BRAdmin Light。
  • 如果您正在使用 Windows® 防火墙，或者反间谍软件或反病毒应用程序的防火墙功能，请暂时禁用防火墙。确认可以进行打印时，请遵循相关说明配置软件设置。
  • 节点名称：节点名称出现在当前 BRAdmin Light 窗口中。对于有线网络，默认节点名称为 "BRNxxxxxxxxxxxx"；对于无线网络，默认节点名称为 "BRWxxxxxxxxxxxx" (其中，"xxxxxxxxxxxx" 为本设备的 MAC 地址/以太网地址)。
  • 默认情况下不需要密码。如果之前已经设置了密码，请输入您的密码。
  1. 安装 BRAdmin Light。
  2. 启动 BRAdmin Light。
    • (Windows Vista®、Windows® 7、Windows Server® 2003 和 Windows Server® 2008)

      点击 (开始)>所有程序>Fuji Xerox>BRAdmin Light>BRAdmin Light。

    • (Windows® 8、Windows Server® 2012 和 Windows Server® 2012 R2)

      点击 (BRAdmin Light)。

    • (Macintosh)

      请转到 Fuji Xerox 网站 (http://www.fujixeroxprinters.com) 上您的设备型号的下载页面，下载最新版本的 BRAdmin Light。

      选择已下载的文件夹，然后双击 BRAdmin Light.jar 图标。

    BRAdmin Light 将自动搜索设备。
  3. 双击您想要更改其设置的打印服务器。
    • 如果打印服务器设置为出厂默认设置 (如果您未使用 DHCP/BOOTP/RARP 服务器)，设备将在 BRAdmin Light 中显示为 未配置。双击未配置的设备。
    • 通过打印网络配置报告，您可以查找到节点名称和 MAC 地址 (以太网地址)。您也可以使用 Fuji Xerox 设备的操作面板查找 MAC 地址。
  4. 如果之前已经设置了密码，请输入您的密码，然后点击确定。
您现在可以更改网络设置。
