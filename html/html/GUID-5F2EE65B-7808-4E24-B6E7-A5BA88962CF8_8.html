操作面板概述

DocuPrint P268 b/DocuPrint P228 db
DocuPrint P268 d/DocuPrint P268 dw

DocuPrint P268 b/DocuPrint P228 db

本节介绍设备操作面板上的四个发光二极管 (LED) 指示灯和两个按钮。　　　　　　　　　　
  1. 电源开关按钮

    • 按 可打开设备电源。

    • 按住 可关闭设备电源。

  2. 墨粉 指示灯
    墨粉 指示灯指示墨粉即将用尽。
  3. 感光鼓 指示灯
    感光鼓 指示灯指示即将需要更换硒鼓。
  4. 纸张 指示灯
    纸张 指示灯指示存在进纸错误。
  5. 就绪 指示灯
    就绪 指示灯通过闪烁指示设备状态。
  6. 执行按钮
    • 按执行可清除某些错误。
    • 按执行可打印设备内存中剩余的任何数据。
    • 按住执行约两秒钟可取消打印作业。

DocuPrint P268 d/DocuPrint P268 dw

有线型号：

带无线功能的型号：

  1. LCD (液晶显示屏)
    显示相关信息，帮助您设置并使用本设备。

    无线型号：

    四级指示器，用于显示使用无线连接时的无线信号强度。

  2. 电源开关按钮

    • 按 可打开设备电源。

    • 按住 可关闭设备电源。LCD 在变暗之前会显示Shutting Down (正在关机)几秒钟。

  3. 网络按钮

    • 有线型号：

      网络

      按网络可设置有线网络设置。

    • 带无线功能的型号：

      WiFi

      按 WiFi 按钮并在计算机上启动无线安装程序。遵循屏幕提示设置本设备与网络之间的无线连接。

      WiFi 指示灯亮起时，表示本 Fuji Xerox 设备已连接到无线接入点。WiFi 指示灯闪烁时，表示无线连接中断或本设备正在连接到无线接入点。

  4. 功能按钮
    确定

    按此按钮可存储设备设置。更改设置后，设备返回上一级菜单。

    返回
    • 按此按钮可返回上一级菜单。
    • 按此按钮可在设置数字时选择前一个数字。
    或 (+ 或 -)
    • 按相应按钮滚动显示菜单和选项。
    • 按此按钮可输入数字，或者可增大或减小数字。按住 或 可更快速地滚动。出现您需要的数字时，按 确定。
  5. 执行按钮
    • 按此按钮可清除某些错误信息。若要清除所有其他错误，请遵循操作面板提示。
    • 按此按钮可打印设备内存中剩余的任何数据。
    • 按此按钮可选择显示的选项。更改设置后，设备返回准备就绪模式。
    • 按住执行约两秒钟可取消打印作业。
