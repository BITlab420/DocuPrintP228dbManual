使用 USB 接口电缆配置设备的无线网络

相关型号:DocuPrint P268 dw
使用 USB 接口电缆将本 Fuji Xerox 设备连接到您的无线网络。一旦连接完成，移除 USB 接口电缆。
  1. 启动计算机，将随机安装光盘插入 CD-ROM 光驱中。
    • 对于 Windows®，请遵循以下说明。
    • 可从 Fuji Xerox 网站 (http://www.fujixeroxprinters.com) 上您的设备型号的下载页面获取 Macintosh 的打印机驱动程序。请遵循屏幕提示安装打印机驱动程序。
  2. 将自动显示开始屏幕。
    出现型号名称屏幕时，选择本设备；出现语言屏幕时，选择所需语言。
  3. 随即出现随机光盘主菜单。如果您接受许可证协议，请点击安装打印机驱动程序，然后点击是。
    遵循屏幕提示进行操作。

    • (Windows Vista® 和 Windows®7)

      如果 Fuji Xerox 安装屏幕未自动出现，请转到计算机。双击 CD-ROM 图标，然后双击 start.exe。

    • (Windows® 8)

      点击任务栏中的 (文件资源管理器) 图标，然后转到计算机 (这台电脑)。双击 CD-ROM 图标，然后双击 start.exe。

    • 如果显示用户帐户控制屏幕，请执行以下操作中的一项：
      • (Windows Vista®) 点击继续。
      • (Windows® 7/Windows® 8) 点击是。
  4. 选择无线网络连接，然后点击下一步 。
  5. 选择 Fuji Xerox 对等网网络打印机或网络共享打印机，然后点击下一步 。
  6. 选择网络共享打印机时，请选择浏览打印机屏幕上的设备队列，然后点击确定。
  7. 在检测到防火墙/防病毒软件屏幕中选择所需防火墙设置的单选按钮，然后点击下一步 。
  8. 选择是的，我有一根 USB 电缆可用于安装。单选按钮，然后点击下一步 。
  9. 遵循屏幕提示配置设置。
    • 显示可以使用的无线网络屏幕时，如果您的接入点设置为不多址发送 SSID，则您可以通过点击高级按钮来手动添加。遵循屏幕提示输入名称(SSID)。
    • 若出现无线设置失败屏幕，请点击重试，然后重复上述步骤。
    • 设置过程中如果需要联系 Fuji Xerox 呼叫中心寻求帮助，请务必先准备好 SSID (网络名称) 和网络密钥。我们无法帮助您确定此信息。
    • 如果您不确定 SSID 和网络密钥等信息，将无法继续进行无线设置。
      如何查找上述信息
      • 请参见无线接入点/路由器随附的说明书。
      • 初始 SSID 可能是制造商的名称或型号名称。
      • 如果您不确定安全信息，请咨询路由器制造商、系统管理员或互联网供应商商。

完成无线设置后，安装程序将指导您完成驱动程序安装。请点击安装对话框中的下一步 ，然后遵循屏幕提示进行操作。
