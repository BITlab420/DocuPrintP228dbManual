使用 PIN 方式配置 Wi-Fi Direct 网络

相关型号:DocuPrint P268 dw
若您的移动设备支持 Wi-Fi Direct 的 PIN 方式，则可遵循以下步骤配置 Wi-Fi Direct 网络：
  1. 按 或 选择Network (网络)选项，然后按 确定。
  2. 按 或 选择Wi-Fi Direct选项，然后按确定。
  3. 按 或 选择PIN Code (PIN码)选项，然后按确定。
  4. 显示Wi-Fi Direct On? (开启 Wi-Fi Direct？)时，按接受。要取消，请按 。
  5. 设备 LCD 上显示Activate Wi-Fi Direct on other device and press OK. (激活其他设备的 Wi-Fi Direct 功能，并按 OK。)时，激活移动设备上的 Wi-Fi Direct 功能 (有关说明，请参见移动设备的使用说明书)。按本设备上的确定。
    Wi-Fi Direct 设置随即启动。要取消，请按 返回。
  6. 执行以下操作中的一项：

    • 当本 Fuji Xerox 设备为组拥有者 (G/O) 时，本设备将等待您的移动设备发出的连接请求。显示Input PIN Code (输入 PIN 码)时，在本 Fuji Xerox 设备上输入移动设备上显示的 PIN 码。按确定完成设置。

      如果本 Fuji Xerox 设备上显示 PIN，请在移动设备上输入此 PIN。

    • 当本 Fuji Xerox 设备不是 G/O 时，本设备将显示可以用于配置 Wi-Fi Direct 网络的设备名称。按 或 选择您想连接的移动设备，然后按 确定。按Rescan (重新扫描)可重新搜索可用设备，然后转到下一步。
  7. 执行以下操作中的一项：

    • 按 在本 Fuji Xerox 设备上显示 PIN，在移动设备上输入此 PIN，然后转到下一步。

    • 按 在本 Fuji Xerox 设备上输入移动设备上显示的 PIN 码。按 确定，然后转到下一步。

  8. 如果已成功连接移动设备，设备 LCD 上会显示Connected (已连接)。Wi-Fi Direct 网络设置已完成。
