使用网络服务安装用于打印的驱动程序 (Windows Vista®、Windows® 7 和 Windows® 8)

相关型号:DocuPrint P268 d / DocuPrint P268 dw
使用网络服务监控网络上的打印机。
  • 验证主机和 Fuji Xerox 设备是否在同一子网中，或者路由器是否正确配置，以便在两设备之间传输数据。
  • 配置此项设置之前，必须先配置本 Fuji Xerox 设备的 IP 地址。
  • 对于 Windows Server® 2008、Windows Server® 2012 和 Windows Server® 2012 R2，则必须安装打印服务。
  1. 插入随机安装光盘。
  2. 打开 Windows® Explorer，导航至您的 CD-ROM 光驱，然后选择install>driver>gdi>32_64。
  3. 双击 dpinstx64.exe 或 dpinstx86.exe。
    • 如果您使用的是 32 位计算机，请选择 dpinstx86.exe。
    • 如果您使用的是 64 位计算机，请选择 dpinstx64.exe。
  4. 如果显示用户帐户控制屏幕，请执行以下操作中的一项：
    • (Windows Vista®)

      点击继续。

    • (Windows® 7/Windows® 8)

      点击是。

  5. 执行以下操作中的一项：
    • (Windows Vista®)

      点击 (开始)>网络。

      设备的网络服务名称与打印机图标一起显示。

      右击您想安装的设备。

    • (Windows® 7)

      点击(开始)>控制面板>网络和 Internet>查看网络计算机和设备。

      设备的网络服务名称与打印机图标一起显示。

      右击您想安装的设备。

    • (Windows® 8)

      将鼠标移到桌面的右下角。显示菜单栏时，点击设置>更改电脑设置>设备>添加设备。

      随即显示设备的网络服务名称。

    • (Windows® 8.1)

      将鼠标移到桌面的右下角。显示菜单栏时，点击设置>更改电脑设置>电脑和设备>设备>添加设备。

      随即显示设备的网络服务名称。

    • Fuji Xerox 设备的网络服务名称为设备型号名称加设备 MAC 地址 (以太网地址) (例如，FX DocuPrint (型号名称) [XXXXXXXXXXXX] (MAC 地址/以太网地址))。
    • (Windows® 8)

      将鼠标移到设备名称上方即可显示设备信息。

  6. 执行以下操作中的一项：
    • (Windows Vista®/Windows® 7)

      点击设备下拉菜单中的安装。

    • (Windows® 8)

      选择您想安装的设备。
