设置自动断电模式

相关型号:DocuPrint P268 d / DocuPrint P268 dw
如果设备处于深度休眠模式特定时长，基于您的设备型号和设置，将自动进入断电模式。当设备连接至有线网络或无线网络时，将不会进入断电模式。
  • 若要启动打印，按操作面板上的 ，然后发送打印作业。
DocuPrint P268 b/DocuPrint P228 db
DocuPrint P268 d/DocuPrint P268 dw

DocuPrint P268 b/DocuPrint P228 db

  1. 确保已合上前盖且已插入电源线插头。
  2. 按 打开设备电源并确保仅 就绪 指示灯亮起。
  3. 按执行四次开启或关闭此功能。

    自动断电模式开启时，所有指示灯闪烁两次。

    自动断电模式关闭时，所有指示灯闪烁一次。

DocuPrint P268 d/DocuPrint P268 dw

选择设备在进入自动断电模式之前停留在深度休眠模式的时间。

  1. 按 或 选择General Setup (常规设置)选项，然后按确定。
  2. 按 或 选择Ecology (环保)选项，然后按确定。
  3. 按 或 选择Auto Power Off (自动断电)选项，然后按确定。
  4. 按 或 选择Off (关)、1 hour (1 小时)、2 hours (2 小时)、4 hours (4 小时)或8 hours (8 小时)选项，然后按 确定。
