通过网络中的无线局域网接入点路由器连接到计算机 (基础架构模式)

相关型号:DocuPrint P268 dw
此类网络拥有网络核心的无线局域网中央接入点/路由器。接入点可作为通往有线网络的桥梁或网关。当 Fuji Xerox 无线设备 (本设备) 是此网络的一部分时，将通过接入点接收所有打印作业。
  1. 无线局域网接入点/路由器
  2. 无线网络设备 (本设备)
  3. 连接到无线局域网接入点/路由器的无线计算机
  4. 通过网络接口电缆连接到无线局域网接入点/路由器的有线计算机

配置方式

选择以下方法中的一种在无线网络环境中配置 Fuji Xerox 设备。
  • 使用计算机的安装向导进行无线网络配置 (推荐)
  • 使用操作面板的安装向导进行无线网络配置
  • 使用 WPS 或 AOSS™ 进行一键式无线网络配置
  • 使用 WPS 进行 PIN 方式无线网络配置
