使用 APIPA 配置 IP 地址

相关型号:DocuPrint P268 d / DocuPrint P268 dw

Fuji Xerox 打印服务器支持自动专用 IP 寻址 (APIPA) 协议。使用 APIPA，DHCP 客户端可在 DHCP 服务器不可用时自动设置 IP 地址和子网掩码。设备在 169.254.1.0 至 169.254.254.255 范围内选择其自己的 IP 地址。子网掩码自动设置为 255.255.0.0，网关地址设置为 0.0.0.0。

默认状态下，APIPA 协议为启用。若要禁用 APIPA 协议，请使用设备操作面板、BRAdmin 应用程序或网络基本管理。
